# ucollector无礁每日大赛采集器Edge浏览器扩展

### 1.项目介绍

- 借助本工具可以采集每日大赛图片及视频资源到本地，在本地视频播放系统里查看与播放，能够嗅探并下载相关图片。

### 2.项目使用

- 2.1 下载web端程序以及浏览器扩展

- 2.2 部署web端程序（建议在本地环境部署）

- 2.3 安装扩展到浏览器（兼容Edge/Chrome）

- 2.4 安装扩展后点击扩展图标，配置接口信息（填写web端api接口地址用于接收采集资源）

- 2.5 打开每日大赛视频播放详情页，点击悬浮窗自动获取当前页视频及图片资源发送到web端存储

- 2.6 采集的资源存储在web端data.json，仅采集图片资源到本地，视频资源仅采集m3u8源直链

- 2.7 最后访问web端地址就能查看采集到的资源了
